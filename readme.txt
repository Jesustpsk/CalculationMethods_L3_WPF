Данный код реализует метод Гаусса для решения системы линейных уравнений. 
Он состоит из двух основных частей: прямого и обратного хода метода Гаусса.
Прямой ход метода Гаусса выполняет преобразования над системой уравнений таким образом,
чтобы добиться ступенчатого вида матрицы системы. Внутренний цикл пробегает по всем строкам 
под текущей строкой и вычитает из них пропорциональные значения, чтобы получить нули до 
диагонали матрицы. Это делается путем вычисления коэффициента пропорциональности (ratio) и 
применения его к соответствующим элементам матрицы.
Обратный ход метода Гаусса вычисляет значение переменных, начиная с последней строки. 
Значение переменной i вычисляется как отношение правой части уравнения к соответствующему 
элементу матрицы на диагонали. Затем происходит обратное вычитание новых значений переменных 
в уравнениях выше по строкам.
